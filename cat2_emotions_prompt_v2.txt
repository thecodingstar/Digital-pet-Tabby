------------------------------------------------------------------------------
MAIN PROMPT  (paste this; attach Cat-2.webp as the character/style reference)
------------------------------------------------------------------------------
A pixel-art sprite sheet of the SAME single orange tabby cat from the reference
image, cloned in the EXACT same art style across every cell. This is ONE
specific cat — keep it perfectly consistent; only the POSE and FACIAL
EXPRESSION change per cell.

Lock these character details in EVERY cell (do not drift, do not simplify):
 - Warm orange fur as the base coat.
 - Darker orange TABBY STRIPES on the back, head and tail.
 - CREAM / pale-belly markings: cream muzzle, cream chest, cream paws/socks,
   cream inner ears. (Critical — v1 wrongly made the cat solid orange.)
 - Thick, even, dark navy-blue outline around the whole body.
 - Same eye style and same chunky 16-bit pixel resolution as the reference.

Composition / layout (must be clean and uniform for slicing):
 - Grid of 4 columns x 4 rows = 16 cells, in the order listed below.
 - Every cell is the SAME square size with EQUAL padding/gutters between cells.
 - One cat per cell, fully inside its cell, never touching or overlapping
   neighbors or cell edges.
 - Every cat is the SAME scale and stands on the SAME baseline (feet aligned),
   so a side profile looks like it's standing on a common floor line.
 - ALL cats FACE THE SAME DIRECTION — facing RIGHT — in a consistent
   side / three-quarter view. (v1 mixed left and right; do not.)
 - TRANSPARENT background (alpha). No white fill, no gradient, no ground
   shadow, no text, no labels, no watermark.

States, one per cell, left-to-right, top-to-bottom:

ROW 1  (motion)
 1. IDLE        - standing calm, neutral face, tail resting low.
 2. WALK A      - mid-stride, front+back legs spread apart, tail level.
 3. WALK B      - opposite stride, legs gathered under body.
 4. SITTING     - sitting upright, tail curled around front paws, content.

ROW 2  (positive)
 5. SLEEPING    - curled in a loaf, eyes closed, small "Zzz" floating above.
 6. HAPPY       - tail straight up, bright eyes, open smiling mouth.
 7. LOVE        - heart-shaped eyes, a couple tiny floating hearts, blissful.
 8. THINKING    - head tilted up, one front paw raised, small "?" above head.

ROW 3  (negative / intense)
 9. ANGRY       - arched back, bristled fur, ears flat back, hissing mouth.
10. HUNGRY      - crouched low, drooling, big pleading eyes, licking lips.
11. SAD         - ears drooped, big watery eyes, two blue tear drops.
12. SCARED      - startled jump, wide round eyes, "!" above head.

ROW 4  (attention / misc)
13. ALERT       - ears perked up, head high, attentive, staring ahead.
14. PLAYFUL     - crouched pounce, rear raised, one paw batting forward.
15. STRETCH     - front legs stretched forward, back arched, wide yawn.
16. PROUD/DONE  - sitting tall, eyes happily closed, smug little smile.

------------------------------------------------------------------------------
HARD CONSTRAINTS  (do NOT drift)
------------------------------------------------------------------------------
- 16 identical-character clones — same breed, same colors, same markings,
  same outline, same size in all cells. Only pose + expression vary.
- KEEP the cream muzzle/chest/paws and the tabby stripes in every cell.
- Crisp hard pixels: NO anti-aliasing, NO smooth/soft shading, NO blur,
  NO gradients, sharp pixel edges only.
- Same pixel-block size as the reference (do not shrink the cat into tiny
  noisy pixels — fill the cell, leave a small even margin).
- Uniform 4x4 grid, equal gutters, transparent background.
- All cats face RIGHT, feet on a shared baseline.

------------------------------------------------------------------------------
NEGATIVE PROMPT
------------------------------------------------------------------------------
blurry, anti-aliased, soft shading, gradient, glow, 3d render, realistic fur,
solid-orange cat with no cream markings, missing stripes, different cat breeds,
color shifts between cells, inconsistent or thin outline, cats facing different
directions, varying sizes, misaligned baseline, drop shadows, white background,
background scenery, text, labels, watermark, signature, jpeg artifacts,
multiple cats per cell, cats touching or cropped at cell edges.
