A pixel-art sprite sheet of the SAME orange tabby cat shown in the reference

Render these states, one per cell:

------------------------------------------------------------------------------
HARD CONSTRAINTS (keep, do not drift)
------------------------------------------------------------------------------
- Identical character in all 16 cells (clone, not variations of breed/color).
- Same palette as reference: warm orange fur, darker orange stripes, cream
  underside, dark navy outline, small white eye glints.
- Same pixel grid size / blockiness as the reference — crisp hard pixels,
  NO anti-aliasing, NO smooth shading, NO soft edges.
- One cat per cell, fully inside its cell, not overlapping neighbors.
- Consistent scale: every cat roughly the same size cell-to-cell.
- Background flat white or transparent only.
